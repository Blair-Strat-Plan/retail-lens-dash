# Retail-Lens
Portfolio Intelligence Suite for Retail CFOs

Retail-Lens is a financial insight dashboard built in Streamlit.  
It helps separate activity (revenue) from value (profit) to support strategic decision-making in a tech hardware retail business.

## Key Insight Areas
- Portfolio Overview
- Revenue vs Profit Contribution
- Capital Efficiency
- Pricing & Margin Discipline
- Concentration Risk
- Tier Structure
- Unit Economics
- Volume vs Margin Tradeoff
- Category Performance
- Exit Candidates

## Tech Stack
- Python
- Streamlit
- Pandas
- Plotly

## Run Locally

1. Create virtual environment  
2. Install dependencies  
   `pip install -r requirements.txt`
3. Run:
   `streamlit run app.py`
